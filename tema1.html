  • Ques es CSS

    • ¿Qué es una etiqueta HTML?

    Durante su evolución, el lenguaje de marcado HTML permitió un cierto grado de personalización. Desde poner texto en negrita, cambiar colores o alinear fotos, casi todo se hace a mano, agregando notas línea por línea que ayudan a personalizar.
    Esto hace que cualquier proyecto web sea difícil de mantener en el tiempo. Entre otras razones, la razón principal es que cualquier cambio en el estilo del sitio web debe copiarse manualmente en todos los elementos relevantes casi línea por línea. Solo pensarlo me hace sudar, pero fue así hace unos años.
    Para que sirve
    El código CSS facilita la vida de los desarrolladores front-end al separar la estructura de un documento HTML de su presentación. En otras palabras, HTML actuará como el esqueleto de la Web, definiendo su estructura básica, mientras que CSS agregará toda la capa de personalización sobre la que la Web define su apariencia final.
    Siguiendo esta base, los diseñadores web pueden cambiar muy fácilmente la apariencia de un sitio web sin afectar significativamente su contenido. El contenido es siempre el mismo, solo cambia con lo que vemos. CSS es fácil de entender y aprender, y nos brinda un control efectivo sobre cómo se diseñan los documentos HTML.
    Por ejemplo, con CSS, podemos definir todas las etiquetas de texto h1 para cambiar su apariencia, cambiar colores, fuentes y agregar efectos al marcado que definimos, sin modificar nada subyacente. Ahora imagina si tuvieras que revisar línea por línea.

  • Como funciona
    • Al contrario de lo que algunas personas piensan, CSS no es un lenguaje de programación. En realidad interactúa con la web, pero colabora con otros lenguajes de programación.
    CSS funciona de una manera muy simple. Los navegadores web modifican la forma en que se nos presentan al aplicar reglas CSS a los documentos. Todo sucede de forma transparente para el usuario. Estas reglas constan de dos elementos clave. El primero es un conjunto de atributos con valores que actualizan la representación del contenido HTML. El segundo elemento es el selector. Estos nos ayudan a seleccionar elementos que se ven afectados por el nuevo valor del atributo. De esta forma, evitamos que CSS afecte a todos los elementos de HTML, y solo afecte a aquellos elementos que queramos. Tales como elementos contenidos en el primer párrafo de nuestro documento o seleccionando alternativamente las filas de una tabla.

  • Framework


    • En resumen, un marco es una estructura previa que se puede utilizar para desarrollar un proyecto.
    Un framework es una plantilla, un esquema conceptual que simplifica el desarrollo de una tarea porque solo necesita ser complementado de acuerdo al trabajo a realizar.
    Si bien su uso más común es en informática, el concepto también se usa en marketing.
    Facilita mucho el trabajo en ambos campos, ya sea desarrollo de software o estrategia de marketing.
  • Para que sirve

    • Los frameworks se utilizan para llevar a cabo proyectos de manera rápida y eficiente con un código más limpio y consistente en el mundo de la programación en menos tiempo. El marco proporciona una infraestructura que los programadores pueden complementar o modificar según sus objetivos.

    • Te permite realizar trabajos en menos tiempo de lo normal.
